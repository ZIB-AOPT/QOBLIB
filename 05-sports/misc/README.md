# Miscellaneous

This directory contains utility scripts for converting sports scheduling instances and solutions between different formats.

## Scripts

### convert_sol2xml.sh

Batch converts all solution files in a directory from `.sol` format to ITC (International Timetabling Competition) `.xml` format.

**Usage:**
```bash
./convert_sol2xml.sh <path_to_directory>
```

- `path_to_directory`: Directory containing `.sol` solution files

**Note:** Uses `sol2itc.py` internally for the conversion.

### convert_xml2lp.sh

Batch converts all `.xml` instance files in a directory to `.lp` (linear program) format using ZIMPL.

**Usage:**
```bash
./convert_xml2lp.sh <path_to_directory>
```

- `path_to_directory`: Directory containing `.xml` instance files

**Requirements:** ZIMPL must be installed and accessible.

**Note:** Uses `itc2mip.py` internally for the conversion.

### itc2mip.py

Converts ITC (International Timetabling Competition) format instances to ZIMPL mathematical programming models.

**Usage:**
```bash
python itc2mip.py --nosoft <file>
```

- `<file>`: ITC format instance file
- `--nosoft`: Excludes soft constraints from the model

**Output:** Prints ZIMPL model to console (redirect to file if needed).

**Example:**
```bash
python itc2mip.py --nosoft instance.xml > model.zpl
```

### sol2itc.py

Converts solution files from ZIMPL/Gurobi `.sol` format to ITC `.xml` format.

**Usage:**
```bash
python sol2itc.py <tbl_file> <file>
```

- `<tbl_file>`: Variable mapping table file (`.tbl` format, generated by ZIMPL)
- `<file>`: Solution file (`.sol` format)

**Output:** Prints XML solution to console (redirect to file if needed).

**Example:**
```bash
python sol2itc.py model.tbl solution.sol > solution.xml
```

## Workflow

A typical workflow for working with ITC sports scheduling instances:

1. **Convert instance to LP:**
   ```bash
   python itc2mip.py --nosoft instance.xml > model.zpl
   zimpl model.zpl
   ```

2. **Solve the LP** (using your preferred solver)

3. **Convert solution back to XML:**
   ```bash
   python sol2itc.py model.tbl solution.sol > solution.xml
   ```
